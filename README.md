🤖 AI Personal Assistant (n8n + LLM Agent)

An agentic AI workflow built using n8n, and OpenRouter (GPT-4o-mini) that automates real-world productivity tasks through natural language commands.

**🔥 Key Capabilities**

📧 Automated email generation & sending (Gmail API)
📅 Smart calendar scheduling with attendees (Google Calendar)
📊 Contact retrieval from Google Sheets (dynamic recipient handling)
🔎 Real-time web search using SerpAPI
🧠 Context-aware conversations using memory buffer
⚡ Multi-step task execution via AI Agent

**🧠 Architecture Highlights**

Built using LangChain Agent inside n8n
Tool-based modular design (Email, Calendar, Search, DB)
Context retention using memory buffer window
Dynamic parameter extraction using AI ($fromAI)
💡 Example Use Cases
“Send an email to my team and schedule a meeting tomorrow”
“Add a calendar event and include all contacts from the sheet”
“Research a topic and summarize findings in an email”

**🛠️ Tech Stack**

n8n (workflow automation)
LangChain Agent
OpenRouter (GPT-4o-mini)
Gmail API
Google Calendar API
Google Sheets API
SerpAPI
